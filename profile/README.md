# SANBI Terrestrial Ecosystem Indicators

This account hosts workflows and data used to produce ecosystem indicators linked to risk assessments (i.e. IUCN Red List of Ecosystems) and protection level assessments.

## Repositories for the National Biodiversity Assessment 2025 (https://nba.sanbi.org.za)

### LCC_terr_RLE
NBA 2025 workflows for various land-cover change analyses and condition analyses used in computation of IUCN RLE Criterion A, B and D assessments

### INV_terr_RLE
NBA 2025 workflows to combine various data sources for invasive alien plants to supplement the land cover chnage and condition analyses in LCC_terr_RLE

### RLE_terr
NBA 2025 workflow for producing terrestrial Red List of Ecosystems indicators

### RLIe_terr
To be added - Red List Index for Ecosystems workflow

### LCC_terr_PROV
To be added - Workflow for calculating provincial statistics on land cover change

### EPL_terr
NBA 2025 workflow to compute Ecsystem Protection Level indicator

## Additional repositories for ecosystem indicators that are not part of the NBA 2025 but are required in the State of Environment reporting and SDG reporting

### TEPI_terr
Workflow for the Terrestrial Ecosystem Protection Index

### EAI_terr
Workflow for the Ecosystem Area Index 

## Repos for spatial dataset preparation

### LCC_terr_8class
Workflow to combine all the National Invasive Alien Plant Survey 2024 data with the 2022 Land Cover - to make an 8 class land cover. This is free to use as it contains no embargoed information.
