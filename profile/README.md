# SANBI Terrestrial Ecosystem Indicators

This account hosts workflows and data used to produce ecosystem indicators linked to risk assessments (i.e. IUCN Red List of Ecosystems) and protection level assessments.

## Repositories for the National Biodiversity Assessment 2025 (<https://nba.sanbi.org.za>)

### [LCC_terr_RLE](https://github.com/SANBI-TER-ECO/INV_terr_RLE)

NBA 2025 workflows for various land-cover change analyses and condition analyses used in computation of IUCN terrestrial Red List of Ecosystems - Criteria A, B and D.

### [INV_terr_RLE](https://github.com/SANBI-TER-ECO/INV_terr_RLE)

NBA 2025 workflows to combine various data sources for invasive alien plants to supplement the land cover change and condition analyses in LCC_terr_RLE.

### [RLE_terr](https://github.com/SANBI-TER-ECO/RLE_terr)

NBA 2025 workflow for that compiles the terrestrial Red List of Ecosystems.

### RLIe_terr

To be added - Red List Index for Ecosystems workflow

### LCC_terr_PROV

To be added - Workflow for calculating provincial statistics on land cover change

### [EPL_terr](https://github.com/SANBI-TER-ECO/EPL_terr)

NBA 2025 workflow to compute the terrestrial Ecosystem Protection Level indicator.

## Repositories for additional ecosystem indicators

### [TEPI_terr](https://github.com/SANBI-TER-ECO/TEPI_terr)

Workflow for the Terrestrial Ecosystem Protection Index which tracks protected areas ad ecosystem representation over time.

### [EAI_terr](https://github.com/SANBI-TER-ECO/EAI_terr)

Workflow for the Ecosystem Area Index which tracks terrestrial ecosystem extent change over time.

## Repositories for spatial data preparation

### [LCC_terr_8class](https://github.com/SANBI-TER-ECO/LCC_terr_8class)

Workflow to combine the National Invasive Alien Plant Survey 2024 data with the 2022 National Land Cover - to make an 8 class land cover. This is free to use as it contains no embargoed information.
